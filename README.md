Thought Journal

A command-line personal journaling application built in Python.

 Features:
- Save thoughts with timestamps
- Mood tracking (1-10)
- Custom tags
- Search entries
- Edit and delete entries
- Weekly summaries
- Basic statistics

Technologies:
- Python
- JSON
- Datetime
- Collections

Future Improvements:
- SQLite database
- GUI version
- Voice input
- AI-generated summaries
